### 🧠 Major System Upgrade — JSON Architecture

Converted game from hardcoded logic → modular system:

#### ✅ Created:
- `/play/data/curiosity.json`
- `/play/data/temple.json`
- `/play/data/sayings.json`

#### ✅ Implemented:
- Dynamic JSON loading via `fetch()`
- Randomized encounter selection
- Flexible reward system
- Perspective unlock system
- Rotating “Saying” system (inspired by Satoshium)

👉 This is a **critical milestone**
The game is now:
> **Data-driven, scalable, and expandable**
